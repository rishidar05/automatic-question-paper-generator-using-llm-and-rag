# ExamGen AI - Intelligent Question Paper Generator

ExamGen AI is a modern, sci-fi themed web application that leverages advanced AI (Groq/Llama-3) to generate expert-quality question papers from any syllabus. It features a robust authentication system, persistent history, interactive verified solving, and a "Big 5" feature pack including rich formatting and branding.

## 🚀 Features

### Core Functionality
-   **AI Generation**: Instantly generate up to **100 questions** based on any syllabus topic.
-   **Structured Outputs**: JSON-based generation ensures perfect formatting for MCQs, Short, and Long answers.
-   **PDF Export**: Download beautifully formatted question papers as PDFs.

### The "Big 5" Feature Pack
1.  **Smart Difficulty**: Choose between *Easy*, *Medium*, and *Hard* difficulty levels.
2.  **Question Types**: specificy *MCQ Only*, *Short Answer*, *Long Answer*, or a *Mix*.
3.  **Answer Keys**: Toggle "Show Answers" to reveal correct answers and marking schemes.
4.  **Edit Mode**: Fully editable papers - fix typos or customize questions before exporting.
5.  **Branding**: Add your **School Name** and **Exam Date** to the PDF header.

### Advanced Tools
-   **Smart Solver**: A dedicated "Solver" page where you can paste any question (or an entire paper) to get detailed AI solutions.
-   **History**: Automatically saves all generated papers. View, reprint, or solve past papers at any time.
-   **Real verification**: Secure Signup/Login with real Email OTP verification (via Brevo).

### UI/UX
-   **Sci-Fi Aesthetic**: Glassmorphism, particle animations, and a "cool" dark mode interface.
-   **High Contrast**: Accessible input fields and responsive design.

## 🛠️ Tech Stack

-   **Frontend**: React.js, Vite, Framer Motion (Animations), Lucide React (Icons), HTML2PDF.
-   **Backend**: Node.js, Express.js.
-   **Database**: SQLite (File-based, zero configuration).
-   **AI Engine**: Groq SDK (Llama-3.3-70b-versatile).
-   **Login**: JWT (JSON Web Tokens), BCrypt (Password Hashing), Brevo (Email OTP).

## 📖 Usage Guide

1.  **Login**: Sign up with a real email. Check your inbox for the OTP code to verify and log in.
2.  **Generator**: 
    -   Enter your **Syllabus** (e.g., "Physics: Kinematics, Newton's Laws").
    -   Select **Count**, **Difficulty**, and **Type**.
    -   Click **Generate**.
3.  **Paper View**:
    -   Review the generated questions.
    -   Click **Edit Paper** to modify text or add School Name.
    -   Click **PDF** to download.
    -   Click **Show Answers** to see the key.
4.  **Solver**: Switch to the "Solver" tab to paste separate doubts or questions for instant AI explanation.
5.  **History**: Access your previously generated papers effectively.

## 🔌 API Endpoints

| Method | Endpoint | Description | Login Required |
| :--- | :--- | :--- | :--- |
| `POST` | `/api/signup` | Create new account | No |
| `POST` | `/api/login` | Login user | No |
| `POST` | `/api/verify-otp` | Verify email OTP | No |
| `POST` | `/api/generate-paper` | Generate Question Paper | **Yes** |
| `POST` | `/api/solve` | Solve arbitrary text/questions | **Yes** |
| `GET` | `/api/history` | Get user's past papers | **Yes** |
